Steps to generate 4 random password

  1. Increase or decrease the default lenght of the password by clicking 'Increase' or 'Decrease' button.
  2. Click on 'Generate Password' button to generate 4 random passwords.
  3. Backspace the default text in the Input box below 'Generate Password' button and fill it with one of the 4 randomly generated password.
  4. If you want to save the passwords in your system's clipboard click on 'Copy to clipboard' button.
  5. Note: After setting the lenght of the password and clicking 'Generate Password' button you cannot click the button again and again
    and should refresh the webpage to get new set of passwords. Sorry!
